# iF.SVNAdmin V2 (Rebuild)

## Setup

Checkout sources:

```bash
git clone https://github.com/mfreiholz/iF.SVNAdmin.git
git checkout svnadmin-v2
git submodule init
git submodule update
```

Set directory permissions of `data` directory to 777:

```bash
chmod -R 777 data/
```
